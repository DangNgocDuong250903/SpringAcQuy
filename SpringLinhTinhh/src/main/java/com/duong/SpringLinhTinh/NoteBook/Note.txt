Trên Windows:
Mở Command Prompt hoặc PowerShell với quyền quản trị.
Chạy lệnh netstat -ano | findstr :8080 để tìm PID của tiến trình sử dụng cổng 8080.
Sử dụng lệnh taskkill /PID <PID> /F để dừng tiến trình đó, thay <PID> bằng giá trị bạn tìm được.
